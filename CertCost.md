# IT Certification Costs and Links

This document provides the estimated costs and official links for popular entry-level IT certifications. Please note that costs are subject to change and may vary by region.

## Foundational Certifications

### CompTIA A+
*   **Cost:** $253 USD per exam (two exams, Core 1 and Core 2, are required for a total of $506 USD).
*   **Link:** [CompTIA A+ Certification](https://www.comptia.org/certifications/a)

### Google IT Support Professional Certificate
*   **Cost:** $49 USD per month via Coursera subscription. The total cost depends on the time taken to complete the certificate.
*   **Link:** [Google IT Support Professional Certificate on Coursera](https://www.coursera.org/professional-certificates/google-it-support)

### CompTIA ITF+ (IT Fundamentals)
*   **Cost:** $134 USD.
*   **Link:** [CompTIA ITF+ Certification](https://www.comptia.org/certifications/it-fundamentals)

## Networking Certifications

### CompTIA Network+
*   **Cost:** $358 USD.
*   **Link:** [CompTIA Network+ Certification](https://www.comptia.org/certifications/network)

### Cisco Certified Network Associate (CCNA)
*   **Cost:** $300 USD.
*   **Link:** [Cisco CCNA Certification](https://www.cisco.com/c/en/us/training-events/training-certifications/certifications/associate/ccna.html)

## Security Certifications

### CompTIA Security+
*   **Cost:** $425 USD.
*   **Link:** [CompTIA Security+ Certification](https://www.comptia.org/certifications/security)

## Cloud Certifications

### AWS Certified Cloud Practitioner
*   **Cost:** $100 USD.
*   **Link:** [AWS Certified Cloud Practitioner Certification](https://aws.amazon.com/certification/certified-cloud-practitioner/)

### Microsoft Certified: Azure Fundamentals (AZ-900)
*   **Cost:** $99 USD.
*   **Link:** [Microsoft Certified: Azure Fundamentals (AZ-900)](https://learn.microsoft.com/en-us/certifications/exams/az-900)

## Programming Certifications

*   **Link:** [PCEP – Certified Entry-Level Python Programmer Certification](https://pythoninstitute.org/pcep)
